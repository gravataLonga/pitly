## Install. 

 - git clone https://github.com/gravataLonga/pitly.git  
 - cd pitly  
 - composer install  
 - npm install  
 - npm run dev  
 - Configure de .env file  
    - You may configure STRIPE_KEY and STRIPE_SECRET, to take advantage of purchase.
